# Wedding Template

This is the wedding template project containing the template pages and components for customizable wedding websites.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the template.

## Project Structure

- `/src/app/wedding-template/` - Template pages
- `/src/components/wedding-template/` - Template components
- `/public/` - Static assets
